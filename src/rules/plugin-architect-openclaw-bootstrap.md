## Plugin Architect (available)

plugin-architect is installed in this workspace. Use it to generate a custom,
codebase-focused plugin for any AI coding tool.

To start: say "build plugin" or "generate plugin". The full workflow lives at:

  plugins/core/skills/build-plugin/SKILL.md

All skills: build-plugin, blueprint-select, skeleton-build, skill-create,
agent-create, plugin-validate, git-detect, schema-scan, tech-stack-detect.

Read the relevant SKILL.md file before executing any skill.
